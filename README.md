# KeeBook - React Social Media Dashboard 📝

**KeeBook** is a dynamic React application that allows users to create, view, and manage social media posts. It features a modern UI built with Bootstrap and a robust state management system using React Context and Reducers.

🚀 **Live Demo:** https://kushal4175.github.io/Social-Media/

## ✨ Features

- **Create Posts:** Add new posts with a title, body, reactions, and hashtags.
- **Dynamic Post List:** View all posts in a responsive card layout.
- **Delete Functionality:** Remove posts instantly with a single click.
- **Interactive Sidebar:** Toggle between the Home feed and the Create Post form.
- **Global State:** Uses `useContext` and `useReducer` for clean, centralized data management.
- **Responsive Design:** Fully styled with Bootstrap 5 and custom CSS.

## 🛠️ Tech Stack

- **Frontend:** React.js (Vite)
- **Styling:** Bootstrap 5 & React Icons
- **State Management:** Context API & useReducer
- **Deployment:** GitHub Pages
